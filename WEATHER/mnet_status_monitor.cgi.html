MesoWest Mesonet Status Monitor

The following table displays monitoring statistics from each individual mesonet. Column headers refer to the following...

MesoWest ID: The mesonet ID utilized by MesoWest.
Acronym: The mesonet shortname/acronym utilized by MesoWest, clicking on an acronym will display a graph of mesonet status over the previous 5 days.
Full Name: The mesonet full name utilized by MesoWest, clicking on the full name will display daily station status statistics for all stations in that mesonet.
Reporting: The total number of stations in a mesonet that reported at least once within the "Time Period Checked".
Active: The total number of stations in a mesonet that have reported at least once within the last 30 days.
Reporting %: The fraction of "Reporting Stations" divided by "Active Stations", converted to a percentage value.
Period Checked: The time period of record examined before the time listed by "Latest Update" at the top of the table. Note that this varies per mesonet.

Row colors for the table refer to the following thresholds...

Reporting Percentage: <= 25% Reporting Percentage: 25% - 75% Reporting Percentage: >= 75% New mesonet(s) or new station(s) added

Please keep in mind that these are current statistics, so if a mesonet is providing data with greater than expected latency (greater than the "Period Checked"), it will appear offline in the table below.
To check this, simply click on the mesonet acronym in the table to view daily station status, or go to any Station Interface page to view the latest observations MesoWest has acquired.

Clicking on a column header name will sort the table by the selected column.
